# Local Food Wastage Management System

## Problem Statement

Food wastage is a significant issue, with many households and restaurants discarding surplus food while numerous people struggle with food insecurity.

This project aims to develop a **Local Food Wastage Management System** where:

- Restaurants and individuals can list surplus food.
- NGOs or individuals in need can claim the food.
- SQLite stores available food details and locations.
- A Streamlit application enables interaction, filtering, CRUD operations, analytics, and data visualization.

The goal is to reduce food wastage by efficiently connecting food providers with food receivers within local communities.

## Features

### Overview Dashboard
- Displays key performance indicators (KPIs)
- Shows project insights
- Displays recommendations

### Food Explorer
- Filter food listings by:
  - City
  - Food Type
  - Meal Type
  - Provider Type
- View provider contact information

### Analytics Dashboard
- Interactive charts and visualizations
- Food availability analysis
- Provider and receiver analysis
- Claim status analysis

### SQL Analysis
- Displays outputs of 20 SQL analytical queries

### CRUD Operations
- Add food listings
- Update food quantities
- Delete food listings

---

## Project Workflow

1. Import CSV datasets
2. Load data into SQLite database
3. Perform SQL analysis
4. Build interactive Streamlit dashboard
5. Create visualizations
6. Implement CRUD operations
7. Deploy application using Streamlit Cloud
